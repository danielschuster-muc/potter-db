# Chapters
Esse recurso fornece informações sobre capítulos de um livro específico do universo Harry Potter.

## Attributes
{
<table>
  <thead>
    <tr>
      <th>Nome</th>
      <th>Tipo</th>
      <th>Descrição</th>
      <th>Filtrável</th>
    </tr>
  </thead>
  <tbody>
    {[
      {
        name: "order",
        type: "integer",
        description: "A ordem cronológica desse capítulo dentro do livro.",
        filterable: true,
      },
      {
        name: "slug",
        type: "string",
        description: "Um identificador para esse capítulo.",
        filterable: false,
      },
      {
        name: "summary",
        type: "string",
        description: "O resumo desse capíitulo.",
        filterable: true,
      },
      {
        name: "title",
        type: "string",
        description: "O título desse capítulo.",
        filterable: true,
      },
    ].map((item) => (
      <tr key={item.name}>
        <td>{item.name}</td>
        <td>{item.type}</td>
        <td>{item.description}</td>
        <td>{item.filterable == true ? "✅" : "❌"}</td>
      </tr>
    ))}
  </tbody>
</table>
}
